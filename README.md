# Кастомизация виджетов
- [chart-color-script.js](chart-color-script.js) — окрашивание подписей (меток) значений маркеров
- [chart-shrink-columns.js](chart-shrink-columns.js) — уменьшение ширины столбцов гистограммы
- [chart-monthSorter-script.js](chart-monthSorter-script.js) — сортировка по месяцам для одной серии (при текстовом значении месяца)
- [chart-monthSorter-script-2series.js](chart-monthSorter-script-2series.js) — сортировка по месяцам для двух серий (при текстовом значении месяца)
